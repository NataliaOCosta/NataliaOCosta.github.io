PROJETO JAVA TESTE


O projeto é um sistema de gerenciamento de eventos em Java, com as seguintes características:

Classes Principais:

Main: A classe principal do programa, que cria instâncias de EventManager e UserController para gerenciar eventos e usuários.

EventManager: Responsável por criar, listar e gerenciar eventos, bem como associar usuários a eventos.

Event: Representa eventos com detalhes como nome, endereço, categoria, data e hora, descrição e lista de participantes.

User: Representa usuários com nome de usuário, cidade e lista de eventos nos quais estão inscritos.

UserController: Atua como uma interface entre os usuários e o sistema, fornecendo métodos para criar eventos, registrar usuários e inscrevê-los em eventos.

Funcionalidades: Possibilita a criação de eventos com informações detalhadas.

Permite o registro de usuários no sistema.

Permite que os usuários se inscrevam em eventos.

Lista os eventos disponíveis para os usuários verem detalhes e participantes.

Oferece um menu interativo para os usuários escolherem entre listar eventos, se inscrever em eventos ou sair do programa.

Em resumo, o projeto é uma aplicação simples de gerenciamento de eventos e usuários em Java, que permite interações como criação de eventos, registro de usuários, inscrição em eventos e visualização de informações sobre eventos e seus participantes.
