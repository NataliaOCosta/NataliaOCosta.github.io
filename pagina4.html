CARDÁPIO HTML E CSS

Este projeto HTML é uma página de cardápio virtual para um restaurante de hambúrgueres chamado "SEU MELHOR HAMBURGUER". A página apresenta uma variedade de hambúrgueres e bebidas disponíveis para pedido, juntamente com informações sobre preços e ingredientes.

Principais elementos e seções:

Header: O cabeçalho da página exibe o nome "SEU MELHOR HAMBURGUER" em destaque.

Botão de Pedido: Um botão "PEÇA JÁ" que redireciona os usuários para um serviço de entrega de comida (iFood).

Seção de Lanches: Informações sobre os lanches disponíveis, incluindo nome, preço e ingredientes. Três lanches são apresentados: "Bacon especial," "Hot especial," e "Cheddar especial."

Seção de Bebidas: Informações sobre as bebidas disponíveis, incluindo nome e preço. Quatro bebidas são apresentadas: "Coca-cola 600ml," "Coca-cola 1,5L," "Água," e "Água com gás."

Footer (Rodapé): O rodapé inclui informações de contato, créditos de imagens e links para as redes sociais (Instagram e WhatsApp). Também exibe o ano de direitos autorais (2023).

Estilos e Design:

O projeto utiliza estilos CSS externos para a formatação e layout da página. Usa ícones da fonte Awesome (FontAwesome) para os ícones das redes sociais. Este projeto é uma página de cardápio de restaurante simples e informativa que permite aos clientes ver as opções disponíveis e direcioná-los para fazer um pedido online. É uma forma eficaz de apresentar o menu do restaurante na web.
